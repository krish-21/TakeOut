# TakeOut

Delicious Food, Delivered to You

## Features

- View list of Food which can be ordered
- Add Food to Cart
- View Cart
- Modify Cart within the Cart
  - Remove Food Item
  - Increase amount of Food Item
- Add Delivery Address
- Place Order

## Technology

- Frontend: React.js
- Backend: Firebase
